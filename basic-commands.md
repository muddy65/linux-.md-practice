# Linux Basic Commands
## Goal
Learn the most commonly used linux commands
---
## Commands
|Command | Description|
|----------|-----------|
| pwd | Show current directory|
| ls | List files |
| cd | change Directory |
| mv | move files |
| cp | copy files |
| rm | remove files |

----
## Examples
### Displays Current directory
```bash
pwd
```
### List files
```bash
ls
```
### Copy files
```bash
cp file.txt backup/
```
